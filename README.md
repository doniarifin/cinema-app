## Quick Start

### 1. Rename/copy .env_copy to .env

```
cp .env_copy .env
```

setup your env

### 2. Install depedencies and run the app

- install depedencies

```
go mod tidy
```

- run the app, entry point in `cmd` folder/directory.

```
go run main.go
```
